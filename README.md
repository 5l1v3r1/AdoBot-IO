# AdoBot-IO
AdoBot NodeJS Server with socket.io

# System Requirements
- NodeJS
- MySql DB

# Getting Started
Edit `./config/config.json` to your preferences.

Install dependencies:
```
$ sudo npm install -g sequelize-cli
$ npm install
$ sequelize db:migrate
```

Run local server
```
$ node index.js
```

Browse to [http://127.0.0.1:3000](http://127.0.0.1:3000)
